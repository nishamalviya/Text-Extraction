# Document OCR API
## Run Steps

1. Create and activate a virtual environment.

PowerShell:
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

2. Install dependencies.

```powershell
pip install -r requirements.txt
```


4. Start the API.

```powershell
python main.py
```

5. Open Swagger UI.

```
http://localhost:8000/docs
```
